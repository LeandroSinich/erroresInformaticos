¿Qué tipos de errores existen en un programa informático?

Error de sintaxis

Ocurre simplemente cuando hay un error en la escritura del código.
Este tipo de error no permite la ejecución del código.

Error de Lógica

Se producen cuando la lógica del código es incorrecta para obtener un determinado resultado.
Este tipo de error puede ser difícil de detectar y corregir, ya que puede ocurrir en cualquier parte del código, y no genera error en la consola.

Error en tiempo de Ejecución

Este tipo de error aparece mientras se ejecuta el programa.
Normalmente ocurren cuando se intenta realizar una operación imposible de llevar a cabo. El código se ejecuta normalmete hasta que ocurre dicho error, deteniéndose la ejecución.
